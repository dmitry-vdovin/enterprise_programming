# работа с текстовыми данными в формате XML.

задание: написать программу на языке Java, реализующую некоторые операции над указанным XML-документом. Основные требования:
- графический интерфейс
- проверка на соответствие схеме (.xsd)
- пользовательское меню с пунктами: вывод всех сведений о книгах в виде таблицы, добавление новой книги, поиск книги по автору/году издания/категории, переоценка отдельных книг (изменение цены), эмулирование выдачи книги читателю
