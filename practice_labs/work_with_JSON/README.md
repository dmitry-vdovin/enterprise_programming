# работа с текстовыми данными в формате JSON.

задание: создать класс зачётная книжка (Gradebook) с внутренним классом, с помощью объектов которого можно хранить информацию о сессиях, зачётах, экзаменах. Реализовать тестовую программу для проверки работоспособности приложения. Входной файл - "input.txt", выходной - "output.txt", содержащий в себе информацию о студентах - отличниках, сдавших все экзамены на оценки не ниже 9, а также сдавших все зачёты. Выходной файл содержит следующую информацию о студенте: фамилия, имя, отчество, курс, группа, номер сессии (1-9), название предмета, оценка для экзамена.
